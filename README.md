# NoteFlow — AI Study Buddy 📚
> AI-powered study productivity OS built with React + TypeScript + Gemini API

## Features
- 📁 Smart folder organization for notes
- 🤖 AI Chat powered by Gemini 3.1
- ✍️ Write Canvas — distraction-free writing
- 📄 Built-in PDF Viewer
- ⏱️ Pomodoro Timer
- 🔥 Streak Manager
- 📋 Routine Tracker

## Tech Stack
React · TypeScript · Gemini 3.1 API · Framer Motion · Tailwind CSS · shadcn/ui
